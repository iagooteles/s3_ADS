Para que um projeto de banco de dados represente as reais necessidades do ambiente e do cliente, um modelo (diagrama) de dados deve ser utilizado.
    Descrever formato, relacionamentos, semântica e restrições de um banco;
    Simplifica a compreensão de um sistema, através de uma visão ampla das partes que compõem o negócio;
    ....

Modelo lógico (MER) (Modelo de Entidade Relacionamento) - ;
    - Entidades: ;
        A entidade pode ser concreta(Aquilo que se pode pegar: ALUNO, PROFESSOR) ou abstrada(Não se pode pegar: TURMA, DISCIPLINA, CURSO); 
    - Atributos: Propriedades de uma entidade; o que definem a entidade; Cada entidade tem valores para cada um de seus atributos;
        Composição: Simples ou composto; 
            Simples => Não faz sentido ser desmembrado;
            Composto => Pode ser desmembrado;
            (EX: Nome simples ou composto / Endereço completo ou parcial / Data de nascimento separado em dia mês e ano ou não / )
        Valor: Único valor ou Múltiplo;
            Únicovalorado => Aceita apenas 1 valor;
            Múltiplo => Pode aceitar múltiplos valores;
        Armazenamento: Armazenado ou derivado;
            Armazenado: Valor que não pode ser inferido;
            Derivado: Quando o valor pode ser inferido; EX: Valor total(que seria qnt de produtos * valor ) / Idade (que pode ser derivado da data de nascimento);
    - Relacionamentos: Como atributos e entidades se relacionam;

- Entidade forte: Existe independente de outras entidade;
- Entidade fraca: Sua existência depende de outras entidades;

* Atributo identificador
* Chaves simples - 
* Chaves compostas - Combinação de dois ou mais atributos que os valores deles conseguem identificar cada registro de uma entidade;

Diagramas => 

Retangulos representam entidades;
Elipses representam atributos;
Losangos representam relacionamentos;

O atributo chave tem seu nome sublinhado;
Todo atributo derivado é pontilhado;
Todo atributo multivalorados são respresentados por 2 elipses;
Atributos compostos são representados por segmentos ao seu atributo composto
